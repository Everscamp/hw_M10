Система класів для управління адресною книгою.

Функціональність:

AddressBook:
    Додавання записів.
    Пошук записів за іменем.
    Видалення записів за іменем.
Record:
    Додавання телефонів.
    Видалення телефонів.
    Редагування телефонів.
    Пошук телефону.
